3. 向量 `java.util.Vector`
    - `Vector` 的重要方法
        - add  增加元素
        - get  获得某一个元素
        - size  获得实际存在的元素的个数
        - capacity  获得系统分配的容量大小，不过容量是可变的
        - clear  清除所有的元素    
      - remove  移除某一个索引的元素或移除第一个该元素
      - set 在某一个索引处设置新的值
