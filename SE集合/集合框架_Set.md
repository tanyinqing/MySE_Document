### Set 应有获取一些不重复的值

> 不可重复元素

- `HashSet`运用的还多一些

  > 使用 `HashMap` 存储元素

  > 无序  无法获取某一个值 但可以迭代

  > 效率高
  
  ![HashSet](../image/javase/diagram/HashSet.png)
  
- `LinkedHashSet`

  > 使用 `HashMap` 实现

  > 迭代输出时按元素添加顺序排序
  
  ![LinkedHashSet](../image/javase/diagram/LinkedHashSet.png)
  
- `TreeSet`  

  > 使用 `红-黑 树` 存储元素

  > 按元素值排序 中文的话按编码进行排序
  
  ![TreeSet](../image/javase/diagram/TreeSet.png)
